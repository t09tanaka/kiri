# Git Diff Display Design

## Overview

Display git status and diff information in the file tree and editor.

## Requirements

1. Show git status icons in file tree (modified, added, deleted)
2. Show line-level diff in editor gutter
3. Integration with git2 Rust crate

## Architecture

### Backend (Rust)

Using git2 crate for git operations.

```rust
#[derive(Debug, Clone, Serialize)]
pub struct GitStatus {
    pub path: String,
    pub status: GitFileStatus,
}

#[derive(Debug, Clone, Serialize)]
pub enum GitFileStatus {
    Modified,
    Added,
    Deleted,
    Renamed,
    Untracked,
    Ignored,
    Unchanged,
}

#[tauri::command]
fn get_git_status(repo_path: String) -> Result<Vec<GitStatus>, String>

#[tauri::command]
fn get_file_diff(repo_path: String, file_path: String) -> Result<Vec<DiffHunk>, String>
```

## UI Design

### File Tree

Show status icon next to file name:

- M (yellow): Modified
- A (green): Added
- D (red): Deleted
- U (gray): Untracked
- ? (gray): Ignored

### Editor Gutter

Show diff markers in left gutter:

- Green bar: Added lines
- Red bar: Deleted lines
- Blue bar: Modified lines
